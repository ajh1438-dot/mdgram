# 유저 스토리

## 방문자 (Reader)

### 탐색
- **US-R01**: 방문자로서, 페이지에 들어오면 마인드맵으로 이 사람의 관심사 전체 지도를 한눈에 보고 싶다
- **US-R02**: 방문자로서, 왼쪽 폴더트리에서 관심 있는 주제를 클릭하면 오른쪽에 마크다운 콘텐츠가 나오길 원한다
- **US-R03**: 방문자로서, CSS 테마를 바꿔서 내 취향에 맞는 읽기 환경으로 전환하고 싶다

### 반응
- **US-R04**: 방문자로서, 마인드맵의 키워드를 클릭하면 "이 주제로 대화하고 싶어요" 버튼이 보이길 원한다
- **US-R05**: 방문자로서, 마크다운 본문의 특정 문장을 드래그하면 그 문장에 대한 댓글을 남기고 싶다
- **US-R06**: 방문자로서, 어떤 키워드에 관심이 있어서 연락하는지가 명시된 주제 기반 메시지를 보내고 싶다

### 발견
- **US-R07**: 방문자로서, 다른 방문자들이 어떤 키워드에 가장 많이 반응했는지 보고 싶다
- **US-R08**: 방문자로서, 특정 문장에 달린 다른 사람들의 댓글을 읽으며 "나도 이 생각을 했구나" 느끼고 싶다

## 소유자 (Owner / 안지훈)

### 콘텐츠 관리
- **US-O01**: 소유자로서, 관리자 페이지에서 폴더를 통째로 추가/삭제/이동하고 싶다
- **US-O02**: 소유자로서, 마크다운 파일을 웹 에디터에서 작성하고 실시간 프리뷰로 확인하고 싶다
- **US-O03**: 소유자로서, 폴더트리를 변경하면 마인드맵이 자동으로 업데이트되길 원한다

### 테마/디자인
- **US-O04**: 소유자로서, 여러 CSS 테마 중 선택하거나 커스텀 CSS를 적용하고 싶다

### 소통 관리
- **US-O05**: 소유자로서, 받은 키워드 반응/문장 댓글/주제 메시지를 한 곳에서 관리하고 싶다
- **US-O06**: 소유자로서, 어떤 키워드가 가장 인기 있는지 통계를 보고 싶다
