# 마일스톤

## Phase 1: 코어 퍼블리싱 엔진
> 폴더트리 + 마크다운 리더 + 테마 시스템

### 태스크
- [ ] Next.js 15 프로젝트 셋업 (TypeScript, Tailwind)
- [ ] Supabase 프로젝트 생성 + DB 스키마 설계
- [ ] 폴더트리 데이터 모델 구현 (JSONB)
- [ ] 폴더트리 사이드바 컴포넌트 (접기/펼치기)
- [ ] 마크다운 리더 컴포넌트 (react-markdown + GFM)
- [ ] CSS 테마 시스템 (라이트/다크/포레스트, data-theme 토글)
- [ ] 초기 콘텐츠 시딩 (안지훈 폴더구조 + 마크다운 파일)

### 완료 기준
폴더트리에서 파일 클릭 → 마크다운 렌더링 + 테마 전환 가능

## Phase 2: 마인드맵 + Hero
> 폴더트리에서 자동 생성되는 마인드맵 + 세계관 Hero

### 태스크
- [ ] D3.js 마인드맵 컴포넌트 (폴더구조 → force graph)
- [ ] 마인드맵 ↔ 폴더트리 연동 (노드 클릭 → 파일 이동)
- [ ] Hero 섹션 (타이틀 + 서브타이틀 + 배경 마인드맵)
- [ ] 스크롤 전환 애니메이션 (Hero → 탐색기)
- [ ] 마인드맵 테마 동기화 (CSS Variables 공유)

### 완료 기준
Hero에서 마인드맵이 보이고, 노드 클릭 시 탐색기로 이동

## Phase 3: 소셜 레이어
> 키워드 반응 + 문장 댓글 + 주제 메시지

### 태스크
- [ ] 키워드 반응 버튼 (마인드맵 노드 + 폴더트리 항목)
- [ ] 반응 카운트 저장/표시 (Supabase)
- [ ] 문장 선택 감지 (Web Selection API)
- [ ] 댓글 팝오버 UI + 저장/조회
- [ ] 하이라이트 표시 (댓글 달린 문장)
- [ ] 주제 기반 메시지 모달 (키워드 태그 + 본문)
- [ ] 메시지 저장 + 이메일 알림

### 완료 기준
방문자가 키워드에 반응, 문장에 댓글, 주제 기반 메시지 전송 가능
