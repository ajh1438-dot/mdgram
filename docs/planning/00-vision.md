# 비전: 연결의 숲

## 왜 만드는가

기존 SNS는 "오늘 뭐 먹었어? 어디 갔어?"를 보여준다.
기존 자기소개 페이지는 "나 이런 사람이에요"를 보여준다.

둘 다 **"이 사람과 대화하면 어떤 맛이 날까?"**는 안 보여준다.

연결의 숲은 다르다.
**"나는 이렇게 생각하고, 이렇게 연결합니다. 들어와서 구경해보세요."**

## 무엇을 만드는가

폴더트리 기반 마크다운 퍼블리싱 + 소셜 레이어.

- 콘텐츠는 옵시디언처럼 폴더/파일 구조
- 뷰어는 마크다운 리더 + CSS 테마
- 시각화는 마인드맵 (폴더구조에서 자동 생성)
- 소셜은 키워드 반응 + 문장 댓글 + 주제 기반 메시지

## 성공한다면

방문자가 3분 안에 이렇게 느낀다:

> "오, 나도 이 사람처럼 이 주제에 관심 있었는데.
> 이 내용으로 좀 더 얘기해봐야겠다."

> "이런 생각을 한다고? 독특하고 신선하다.
> 함께 대화 나누고 싶다."

## 장기 비전 (2차 MVP 이후)

1. **20명 전용 비공개 SNS** — 각자 URL 입력 → 자동 마크다운 추출 → 우리끼리 피드
2. **누구나 자기 숲 만들기** — AI가 대화만 먹고 폴더구조 + 마크다운 자동 생성
3. **관심사 기반 매칭** — 키워드 반응 데이터로 "이 사람과 관심사 87% 겹침" 추천
4. **"생각하고 연결하는 사람들의 마을"** — 보기만 하는 SNS가 아니라, 서사로 연결되는 커뮤니티
