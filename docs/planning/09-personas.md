# 페르소나

## Persona 1: 지훈 (소유자 / Creator)

| 항목 | 내용 |
|------|------|
| 이름 | 안지훈 |
| 나이 | 30대 후반 |
| 직업 | CPA, 게임사 경영관리 |
| 기술 수준 | 옵시디언 파워유저, Claude Code로 바이브코딩 |

### 행동 패턴
- 옵시디언에 방대한 노트 보유, 대화로 컨텍스트를 넘기는 방식 선호
- 글쓰기를 귀찮아함 → AI에게 요약/정리 위임
- 관심사를 폴더 구조로 체계화하는 것이 자연스러운 사람

### 니즈
- "내 머릿속 연결을 보여주고 싶다"
- "이 관심사로 대화할 사람을 찾고 싶다"
- 관리자 페이지에서 폴더만 갈아끼면 전체가 바뀌는 편의성

### 페인포인트
- 기존 자기소개 페이지는 한번 만들면 끝 → 살아있는 구조가 아님
- 방명록 외에 구체적인 소통 수단이 없음

## Persona 2: 민수 (방문자 / Reader — 비슷한 관심사)

| 항목 | 내용 |
|------|------|
| 이름 | 김민수 |
| 나이 | 30대 |
| 직업 | 마케터, 독서모임 활동 |
| 기술 수준 | 일반 사용자 |

### 행동 패턴
- 독서모임에서 새로운 사람의 관심사를 미리 파악하고 싶어함
- 책 목록보다 "이 책이 이 사람을 어떻게 바꿨는지"가 궁금
- 공감되는 문장을 발견하면 바로 반응하고 싶음

### 니즈
- "이 사람이 양자역학에도 관심 있어? 나도인데!"
- "이 문장 너무 좋다, 댓글 남기고 싶다"
- 관심사가 겹치면 바로 연락하고 싶다

### 시나리오
1. 마인드맵에서 "건강" 노드 클릭 → "오토파지" 발견
2. 오토파지 파일 읽다가 "인슐린을 줄이기 위해..." 문장 드래그 → 댓글: "저도 이거 해봤는데요!"
3. "건강" 키워드에 "대화하고 싶어요" 클릭
4. 주제 기반 메시지: [건강, 오토파지] "간헐적 단식 경험 공유하고 싶습니다"

## Persona 3: 혜진 (방문자 / Reader — 다른 관심사, 호기심)

| 항목 | 내용 |
|------|------|
| 이름 | 이혜진 |
| 나이 | 20대 후반 |
| 직업 | UX 디자이너, MVP 챌린지 참여자 |
| 기술 수준 | 프론트엔드 기본 가능 |

### 행동 패턴
- 다른 챌린지 멤버들의 페이지를 둘러보며 영감 받기
- "이 사람 되게 독특하다, 회계사인데 양자역학?"이라는 신선함에 끌림
- 자기도 이런 페이지를 만들어보고 싶음

### 니즈
- "어떻게 이렇게 다양한 걸 연결할 수 있지?"
- "나도 내 관심사를 이런 식으로 정리해볼까?"
- 디자인/UX 측면에서 어떻게 구현했는지 궁금

### 시나리오
1. Hero에서 "연결의 숲" 세계관에 호기심 → 스크롤
2. 마인드맵에서 "영성 - 양자역학" 연결을 보고 "세상을 이렇게도 보는구나"
3. "통섭" 키워드에 "대화하고 싶어요" 클릭
4. 테마를 포레스트 모드로 바꿔보면서 "이거 어떻게 만들었지?"
