# 기능 명세

## Feature 1: 폴더트리 마크다운 퍼블리싱

### 설명
옵시디언 스타일 폴더/파일 구조로 콘텐츠를 조직하고, 마크다운 리더로 렌더링

### 상세
- 왼쪽 사이드바: 접기/펼치기 가능한 폴더트리
- 오른쪽 메인: 선택된 마크다운 파일의 렌더링된 내용
- 지원 포맷: 마크다운 (GFM), 이미지 임베드
- 폴더트리 데이터 = Supabase에 JSON 트리 구조로 저장
- 마크다운 콘텐츠 = Supabase Storage 또는 테이블에 저장

### 핵심 제약
- 폴더트리만 갈아끼면 콘텐츠 전체가 바뀌어야 함
- 모바일에서도 폴더트리 탐색 가능 (드로어 방식)

---

## Feature 2: CSS 테마 시스템

### 설명
옵시디언처럼 CSS 테마를 교체하여 전체 페이지 스타일을 변경

### 상세
- CSS Variables 기반 테마 시스템
- 기본 제공 테마: 라이트, 다크, 포레스트(숲 느낌)
- `data-theme` 속성으로 토글
- 마크다운 리더 + 마인드맵이 동시에 테마 반영
- 소유자가 관리자 페이지에서 테마 선택
- 방문자도 테마 전환 가능 (localStorage 저장)

---

## Feature 3: 인터랙티브 마인드맵

### 설명
폴더트리 구조에서 자동 생성되는 인터랙티브 마인드맵

### 상세
- D3.js force graph로 구현
- 중앙 노드: "호기심" (또는 소유자 이름)
- 1레벨 노드: 폴더명 (건강, 과학, 영성, 리더십, 통섭)
- 2레벨 노드: 파일명 (30kg_다이어트, 카를로_로벨리 등)
- 노드 클릭 → 해당 폴더/파일로 이동 (폴더트리와 연동)
- 노드 호버 → 파일 첫 줄 프리뷰 툴팁
- **핵심**: 폴더트리 CRUD 시 마인드맵 자동 반영 (동일 데이터 소스)

---

## Feature 4: 키워드 반응 버튼

### 설명
마인드맵의 키워드(노드)마다 "이 주제로 대화하고 싶어요" 반응 버튼

### 상세
- 마인드맵 노드 클릭 시 팝오버에 "대화하고 싶어요" 버튼
- 클릭 시 반응 카운트 증가 (익명/로그인 구분)
- 소유자 대시보드에서 키워드별 반응 수 확인
- 인기 키워드 하이라이트 (노드 크기 또는 색상 변화)

---

## Feature 5: 문장 선택 댓글

### 설명
마크다운 본문의 특정 문장을 드래그하여 해당 문장에 댓글을 남기는 기능

### 상세
- Web Selection API로 텍스트 범위 감지
- 선택 시 팝오버: "이 문장에 댓글 달기" 버튼
- 댓글 저장: 선택 텍스트 + 위치(파일 경로 + 오프셋) + 내용 + 작성자(익명/닉네임) + 시간
- 마크다운 본문에 댓글 달린 문장 하이라이트 표시
- 하이라이트 클릭 → 댓글 스레드 슬라이드 오픈
- MVP에서는 로그인 없이 닉네임만 입력

---

## Feature 6: 주제 기반 메시지

### 설명
방문자가 특정 키워드/주제에 관심이 있어서 연락한다는 맥락이 보이는 메시지 시스템

### 상세
- 메시지 작성 시 관련 키워드 태그 자동 추천 (현재 보고 있는 폴더/파일 기반)
- 메시지 필드: 키워드 태그 + 본문 + 발신자 닉네임/이메일
- 소유자는 관리자 페이지에서 키워드별로 그룹화된 메시지 확인
- MVP에서는 이메일 알림 연동

---

## Feature 7: 관리자 페이지

### 설명
소유자가 폴더/파일을 자유롭게 관리하고, 댓글/반응/메시지를 확인하는 대시보드

### 상세
- 폴더/파일 CRUD: 생성, 삭제, 이름 변경, 이동 (드래그&드롭)
- 마크다운 에디터: 실시간 프리뷰 (split view)
- 테마 선택/프리뷰
- 댓글 관리: 승인/삭제
- 반응 통계: 키워드별 반응 수 차트
- 메시지 인박스: 키워드 태그로 그룹화

---

## Feature 8: Hero 섹션

### 설명
페이지 첫 화면 — 세계관 전달 + 마인드맵 + 탐색 유도

### 상세
- 타이틀: "연결의 숲" (커스터마이즈 가능)
- 서브타이틀: 소유자의 한 줄 서사
- 배경: 마인드맵이 은은하게 보이며 인터랙티브
- 스크롤 다운 시 폴더트리 + 마크다운 리더로 자연스럽게 전환
- CTA: "숲을 탐색하기" (스크롤 유도)
