# 화면 목록

## Screen 1: 메인 페이지 — Hero

| 항목 | 내용 |
|------|------|
| URL | `/` |
| 목적 | 세계관 전달 + 탐색 유도 |
| 접근 | 모든 방문자 |

### 구성 요소
- **타이틀**: "연결의 숲" (커스터마이즈 가능)
- **서브타이틀**: 소유자 한 줄 서사 ("호기심 천국에 사는 개척정신 회계사")
- **한 줄 카피**: "모든 게 궁금해서, 배우고 바로 연결합니다. 그게 제 인생의 행복입니다."
- **배경**: 마인드맵이 은은하게 인터랙티브하게 표시
- **CTA**: 스크롤 다운 유도 화살표 ("숲을 탐색하기")
- **테마 토글**: 우상단에 테마 전환 버튼

### 전환
- 스크롤 다운 → Screen 2 (탐색기)
- 마인드맵 노드 클릭 → Screen 2 해당 파일로 이동

---

## Screen 2: 메인 페이지 — 탐색기 (폴더트리 + 마크다운 리더)

| 항목 | 내용 |
|------|------|
| URL | `/` (스크롤 하단) 또는 `/#explore` |
| 목적 | 콘텐츠 탐색 + 반응/댓글 |
| 접근 | 모든 방문자 |

### 구성 요소
- **왼쪽 사이드바**: 폴더트리 (접기/펼치기, 파일 클릭으로 콘텐츠 전환)
- **오른쪽 메인**: 마크다운 리더 (선택된 파일 렌더링)
- **마크다운 리더 기능**:
  - 문장 드래그 → 댓글 팝오버
  - 이미 댓글 달린 문장 하이라이트 (클릭 시 스레드 열림)
- **마인드맵 미니뷰**: 우상단 또는 토글로 현재 위치 표시
- **키워드 반응**: 폴더/파일명 옆에 "관심 있어요" 버튼 + 카운트

### 모바일 레이아웃
- 폴더트리: 좌측 드로어 (햄버거 메뉴로 열기)
- 마크다운 리더: 풀스크린

---

## Screen 3: 문장 댓글 패널

| 항목 | 내용 |
|------|------|
| URL | Screen 2 내 오버레이/슬라이드 패널 |
| 목적 | 특정 문장에 대한 댓글 작성/조회 |
| 접근 | 모든 방문자 |

### 구성 요소
- **선택된 문장**: 상단에 하이라이트로 표시
- **댓글 목록**: 시간순, 닉네임 + 내용
- **댓글 작성 폼**: 닉네임(선택) + 내용 + 제출 버튼
- **닫기**: X 버튼 또는 외부 클릭

## Screen 4: 주제 기반 메시지

| 항목 | 내용 |
|------|------|
| URL | Screen 2 내 모달 또는 `/contact` |
| 목적 | 소유자에게 관심 주제 태그가 붙은 메시지 전송 |
| 접근 | 모든 방문자 |

### 구성 요소
- **키워드 태그**: 현재 보고 있는 폴더/파일 기반 자동 추천, 수동 추가 가능
- **메시지 본문**: 텍스트에어리어
- **발신자 정보**: 닉네임 + 이메일 (선택)
- **제출 버튼**

## Screen 5: 관리자 — 폴더/파일 관리

| 항목 | 내용 |
|------|------|
| URL | `/admin` |
| 목적 | 폴더트리 구조 관리 + 마크다운 편집 |
| 접근 | 소유자만 (Supabase Auth) |

### 구성 요소
- **왼쪽**: 폴더트리 (드래그&드롭으로 이동, 우클릭 메뉴로 CRUD)
- **오른쪽**: 마크다운 에디터 (split view — 에디터 | 프리뷰)
- **상단 툴바**: 새 폴더, 새 파일, 삭제, 이름 변경
- **저장**: 자동저장 또는 저장 버튼 → Supabase 업데이트 → 마인드맵 자동 반영

---

## Screen 6: 관리자 — 대시보드

| 항목 | 내용 |
|------|------|
| URL | `/admin/dashboard` |
| 목적 | 반응/댓글/메시지 관리 + 통계 |
| 접근 | 소유자만 |

### 구성 요소
- **키워드 반응 통계**: 키워드별 반응 수 바 차트
- **최근 댓글 목록**: 파일명 + 선택 문장 + 댓글 내용 (승인/삭제)
- **메시지 인박스**: 키워드 태그별 그룹화, 읽음/안읽음 표시
- **테마 설정**: 현재 테마 선택 + CSS 커스텀 에디터

---

## 화면 이동 흐름

```
[Hero] ──스크롤──→ [탐색기 (폴더트리 + 마크다운)]
  │                      │                │
  │                      │                ├── 문장 드래그 → [댓글 패널]
  │                      │                └── 메시지 버튼 → [주제 메시지 모달]
  │                      │
  │                      └── 키워드 클릭 → 반응 카운트 증가
  │
  └── 마인드맵 노드 클릭 → [탐색기 해당 파일]

[관리자 로그인] → [폴더/파일 관리] ←→ [대시보드]
```
