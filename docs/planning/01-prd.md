# PRD: 연결의 숲 (Forest of Connections)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 연결의 숲 (i-ntroduce v2) |
| 한 줄 설명 | 폴더트리 기반 마크다운 퍼블리싱 + 소셜 레이어로 "나의 서사"를 보여주고 대화를 시작하는 플랫폼 |
| 타겟 사용자 | MVP 챌린지 20명 (1차), 관심사 기반 네트워킹을 원하는 사람들 (2차) |
| 핵심 가치 | "이 사람이랑 이 주제로 얘기하고 싶다"는 감정을 만드는 것 |

## 2. 문제 정의

### 기존 SNS의 한계
- X, 인스타, LinkedIn: 순간/외형/광고 중심 → "나 자체"를 설명하지 못함
- 블로그: 시계열 포스팅 → 구조화된 서사가 안 보임
- Obsidian Publish: 개인 지식 베이스 → 소셜 레이어 없음

### 기존 자기소개 페이지의 한계
- wlsvhr, selfservice, KAIROS, Life OS 등: 보기만 하는 페이지
- 방명록이 유일한 소통 수단 → "이 주제로 대화하고 싶다"는 연결이 안 됨

### 핵심 페인포인트
> "이 사람과 대화하면 어떤 맛이 날까?"를 미리 느낄 수 있는 방법이 없다.

## 3. 솔루션

### 핵심 컨셉
**폴더트리 기반 마크다운 퍼블리싱 + 소셜 레이어**

- **콘텐츠** = 폴더트리 (갈아끼면 전체가 바뀜)
- **뷰어** = 옵시디언 스타일 마크다운 리더 (CSS 테마 교체 가능)
- **소셜** = 키워드 반응 + 문장 선택 댓글 + 주제 기반 메시지
- **시각화** = 폴더트리에서 자동 생성되는 마인드맵

### 차별화 포인트
1. 포스팅이 아니라 **퍼블리싱** — 글 하나가 아니라 "나라는 사람 전체"를 구조로 보여줌
2. **보기만 하는 페이지 → 대화가 시작되는 페이지** — 키워드/문장 단위로 반응 가능
3. **폴더트리 = 마인드맵** — 동일 데이터, 자동 동기화

## 4. MVP 범위

### 필수 기능 (Must Have)
- [ ] 폴더트리 + 마크다운 리더 (옵시디언 스타일)
- [ ] CSS 테마 교체 (최소 3개: 라이트/다크/커스텀)
- [ ] 인터랙티브 마인드맵 (폴더구조에서 자동 생성)
- [ ] 키워드별 "대화하고 싶어요" 반응 버튼
- [ ] 문장 선택 댓글 (드래그 → 댓글 팝업)
- [ ] 주제 기반 메시지 (어떤 키워드에 관심 있어서 연락하는지 표시)
- [ ] 관리자 페이지 (폴더/파일 CRUD, 마크다운 에디터)
- [ ] Hero 카피 + 마인드맵 + 폴더트리 조합 첫 화면

### 나중에 (Nice to Have)
- [ ] Google/GitHub 로그인 (20명 제한)
- [ ] 다른 사람 URL 자동 마크다운 추출
- [ ] 클리핑 (남의 문장을 내 볼트에 저장)
- [ ] DM 기능
- [ ] 피드 (여러 사람의 숲을 한 곳에서 탐색)

## 5. 성공 지표

| 지표 | 목표 |
|------|------|
| MVP 챌린지 발표 반응 | "이거 다르다"는 피드백 |
| 키워드 반응 클릭 수 | 방문자 30% 이상이 1개 이상 키워드에 반응 |
| 문장 댓글 수 | 방문자 10% 이상이 1개 이상 문장 댓글 작성 |
| 주제 기반 메시지 | 1주 내 3건 이상의 주제 기반 연락 |

## 6. 디자인 방향

- **벤치마크**: Life OS (세계관 파는 느낌)
- **톤**: 미니멀 + 몰입감 + 탐색하고 싶은 숲
- **핵심**: 다크모드 기본, 글래스 패널, 부드러운 애니메이션
- **폰트**: Pretendard (본문) + JetBrains Mono (코드/마크다운)
